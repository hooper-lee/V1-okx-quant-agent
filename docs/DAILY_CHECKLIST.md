# OKX Quant Agent 每日检查清单

适合你每天打开系统时，快速按顺序检查一遍。

访问地址：

- `http://3.25.117.0:8010/`

## 1 分钟版本

### 1. 看 `总览`

确认这 4 件事：

- 系统在线
- 当前策略对不对
- 当前资金 / 目标资金正常
- 右侧核心数据看板没有明显空白或异常

### 2. 去 `账户`

点击：

- `刷新账户`

确认这 4 件事：

- `source` 正常
- `mode` 是你预期的 `paper` 或 `live`
- `total_equity` 正常
- 资产表有真实数据

如果这里异常，今天先不要交易。

### 3. 去 `自检`

点击：

- `运行自检`

重点看：

- `OKX`
- `LLM`
- `看板快照`

规则：

- `正常`：继续
- `fallback`：谨慎继续
- `失败`：先修问题

### 4. 去 `策略`

点击：

- `刷新分析`

确认这 4 件事：

- 交易对正确
- 周期正确
- 信号正常
- 风控结果正常

### 5. 如果要验证策略，再去 `回测`

点击：

- `运行回测`

主要看：

- 收益率
- 最大回撤
- 交易次数

### 6. 最后才决定要不要下单

必须同时满足：

- 账户正常
- 自检正常
- 当前模式确认无误
- 你知道自己现在是在测试还是实盘

## 遇到异常时怎么处理

### A. 账户显示异常

先查：

- `OKX_USE_PAPER`
- API Key / Secret / Passphrase
- `OKX_ADAPTER`

### B. OKX 失败

先不要下单，先看：

- `账户`
- `自检`
- `配置 -> OKX 诊断`

### C. 分析超时

先查：

- 模型配置
- 模型连接测试
- Base URL

## 你每天最少要点的按钮

按顺序：

1. `账户 -> 刷新账户`
2. `自检 -> 运行自检`
3. `策略 -> 刷新分析`

如果这 3 步都正常，再继续别的操作。
